# Chorus

## Register

product

## Users & purpose

Engineers and researchers running multi-agent reliability workflows. They need to see how a task was planned, which agents competed, and what the harness produced.

## Primary workflow

Paste a natural-language task, inspect the operator map, play gate-by-gate playback, read the winning output and proof.

## Brand personality

Instrumental, precise, calm. A reliability HUD, not a marketing site.

## Anti-references

SaaS cream dashboards, decorative glass cards, gradient hero metrics, modal-first UX for primary content.

## Accessibility

WCAG AA contrast on HUD panels. Keyboard access to Play run and results. Respect `prefers-reduced-motion` for pipeline animation.
